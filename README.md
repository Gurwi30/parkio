# ParkIO

ParkIO è un'applicazione desktop che permette agli utenti di **creare, gestire e interagire con parcheggi su una mappa del mondo reale**. Fornisce un'interfaccia visiva per gestire la disposizione dei parcheggi, gli spazi e le interazioni direttamente sulla mappa, rendendo la gestione dei parcheggi intuitiva e semplice da comprendere.

## Funzionalità

- **Creazione e gestione dei parcheggi** – Aggiungi, aggiorna e rimuovi parcheggi sulla mappa.  
- **Interfaccia mappa interattiva** – Visualizza parcheggi e spazi con interazioni al passaggio del mouse e clic.  
- **Integrazione con mappe reali** – Visualizza i parcheggi posizionati accuratamente sulla mappa.  
- **Componenti UI avanzati** – Componenti personalizzati per un'interazione fluida e una visualizzazione chiara.  
- **Supporto per più operazioni** – Aggiorna i confini dei parcheggi, gestisci gli spazi e gestisci le richieste di input.

## Installazione

1. Clona il repository:  
```bash
git clone https://github.com/Gurwi30/parkio.git
````

2. Apri il progetto nel tuo IDE Java preferito.

3. Compila e avvia il progetto usando Gradle (Kotlin DSL).

## Utilizzo

1. Avvia l'applicazione.
2. Naviga all'interfaccia mappa.
3. Usa la barra degli strumenti per aggiungere nuovi parcheggi.
4. Clicca sui parcheggi per gestire gli spazi e aggiornare le loro proprietà.
5. Passa il mouse sui parcheggi per vedere informazioni rapide e opzioni interattive.
